# Kiosk User Service

Kiosk User Service is an app of compilation of user services.

## Installation

You need [nodejs](https://nodejs.org/en/) and its [npm](https://nodejs.org/en/) in order to deploy this app.

To install the packages:
```bash
npm install
```
## Starting The Service

Run the application:

```bash
npm run dev
```
or
```bash
npm run start
```
